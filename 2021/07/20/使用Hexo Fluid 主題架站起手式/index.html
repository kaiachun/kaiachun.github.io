使用 Hexo Fluid 主題架站起手式

起因

原本的 Blog 主題只是隨便找個能看的，最近上了六角的直播課程，決定要好好做筆記，仔細查了一下覺得 Fluid 這款蠻好看的 (而且有中文)，因此就決定重新改主題囉。

Hexo 介紹

Hexo 是基於 Node.js 所製作的靜態網頁產生工具。
優點

  1. 編譯速度快
  2. 外掛套件多
  3. 支援 Markdown 格式
  4. 支援一鍵佈屬

缺點

  1. 僅支援靜態網頁

來準備開發環境吧

安裝 Node.js

到 Node.js 官網下載軟體並安裝 LTS 版本(長期支援版 Long-term support，較穩定)。

安裝 Git

到 Git 官網下載軟體並安裝。

檢查一下

別急著進到下一步，身為工程師小步測試是很重要滴~

  • 檢查 有跳出版本號就代表成功了~

    此處的 npm 指令會隨著 Node.js 一起安裝喔~

安裝Hexo

  1. 安裝 Hexo
    npm install hexo-cli -g
    記得隨時檢查一下有沒有安裝成功
    hexo -v

  2. 建立 Hexo 初始化資料夾
    以下使用資料夾名稱為 blog
    hexo init [資料夾名稱]
    意即
    hexo init blog

套用 Fluid 主題

  1. 切進資料夾中的themes目錄
    cd blog\themes

  2. 安裝 Fluid 主題
    git clone https://github.com/fluid-dev/hexo-theme-fluid.git

  • 注意此處用 git 抓下來的檔案會在 hexo-theme-fluid 資料夾內，要將資料夾內的檔案都拉到外層

  1. 更改 blog下的 _config.yml 檔
    找到 Extensions 的位置，改為 theme: fluid

  2. 本地端測試
    hexo server 或 hexo s

成功畫面


就可以看到用 Fluid 主題架設的 Blog 啦~
